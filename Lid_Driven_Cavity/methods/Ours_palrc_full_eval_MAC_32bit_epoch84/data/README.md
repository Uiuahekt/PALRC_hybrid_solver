Data file for Ours
==================

Download the data from:

https://drive.google.com/drive/folders/1Zj4vO_sCJQmIEN2RZtiI1ustbW3RxbxZ?usp=sharing

Place the required NetCDF file here:

```text
github_release/methods/Ours_palrc_full_eval_MAC_32bit_epoch84/data/full_trajectories_MAC_32bit.nc
```

The loss scripts read it as:

```text
../data/full_trajectories_MAC_32bit.nc
```
